The bug of setup virtual Ubuntu

安装Ubuntu虚拟机遇到的坑

  • 一开始用VM14.0新建完虚拟机后直接黑屏，也不跳出安装界面，换了VirtualBox后终于可以安装了。
  • 还有旧的镜像也可能会出问题，我就遇到安装过程显示无法写入硬盘，换了个镜像就好了，最好自己下一个，官网就很方便，而网易啊清华的镜像站看着有点懵逼。
  • 一路顺利来到安装的界面，但是，怎么可能让你那么快如愿，弹出了这个东东：

  • 在网上搜了好久，事实上，是我把引导设置错了分区，应该改成/boot的，而且boot100M太小了。
    最后就很开心啦！
